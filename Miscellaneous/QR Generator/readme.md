This python project is a command-line tool for qr generation.  
Steps:  
1. Upload a csv file of the data you wish to create QRs of.  
2. Update the .conf file with paths of logos that you wish to insert in your qr, if any, and the path where you wish to save your generated qrs.  
3. Select the sheet and column for generation.  

Sit back and let the code to the rest.  


Also works for bulk files. avg generation rate: 8 qrs/sec.
